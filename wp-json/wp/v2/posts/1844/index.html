Felicidades a todos los que llegaron hasta esta parte del tutorial, ahora vamos a ver en acci\u00f3n el proyecto final y como puedes ejecutarlo. <\/p>\n\n\n\n

Como saben dentro de los requisitos de este proyecto, era necesario tener un sistema linux ya que el programa utiliza librer\u00edas propias del sistema y en windows habr\u00eda que hacer algunos cambios, en la pr\u00f3xima parte del tutorial explicare como hacer estos cambios para que tambi\u00e9n funcione en windows. Primero que todo, hay que compilar el programa nos dirigimos a la carpeta del proyecto, abrimos una terminal y tecleamos los siguientes comandos. <\/p>\n\n\n\n

$ gcc -o server -lm server.c -lpthread\n$ gcc -o client -lm client.c -lpthread\n<\/pre>\n\n\n\n
Esto utiliza el compilador propio de linux para compilar ambos programas y como resultado obtenemos 2 archivos binarios en el directorio. <\/p>\n\n\n\n
Luego para ejecutarlos usamos los siguientes comandos.  <\/p>\n\n\n\n
$ .\/server <port> \n<\/pre>\n\n\n\n
D\u00f3nde <port> es un puerto cualquiera de la m\u00e1quina, 8000, 5000, 7800, etc. Este puerto no debe estar ocupado por ninguna aplicaci\u00f3n.<\/p>\n\n\n\n
Abrimos otra terminal y ejecutamos el cliente en el mismo puerto del servidor <\/p>\n\n\n\n
$ .\/client <port> <\/pre>\n\n\n\n
De esta manera se pueden abrir hasta 100 terminales para 100 clientes en la sala de chat\n\n<\/p>\n\n\n\n
